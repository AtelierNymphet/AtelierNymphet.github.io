What lives here

Reference, not replacement

The browser is the searchable interface for private TDW sources. The gallery is a visual shell. The timeline is the coverage index. None of these pages replace the source lanes themselves.

  • Browser Authenticated search over source chunks with bounded snippet output and canonical references.
  • Gallery A Three.js presentation surface that frames the same source dataset in a more cinematic layout.
  • Timeline One horizontal line per source lane, with stroke thickness scaled by message count and a month/year axis.
